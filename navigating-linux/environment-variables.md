# Environment Variables

## Current Terminal's Username

```
echo $USER
```

## Present Working Directory

```
echo $PWD
```

## Home Directory

```
echo $HOME
```

## Process ID of Current Shell

```
echo "$$"
```

## Using Environment Variables as a Shortcut

```
export abc=10.10.10.10
ping $abc
```

## View All Environment Variables

```
env
```
